One-line: The signature eyebrow + serif-title (+ lead) block that opens each page section.

```jsx
<SectionHeading eyebrow="Latest news" title="Stories from the farm">
  Every rescue has a story. Here are the latest from our residents.
</SectionHeading>

<SectionHeading eyebrow="Please help" title="Help us help them" onDark align="start" size="lg" />
```

`align`: `center` (default) · `start`. `size`: `sm|md|lg`. Set `onDark` on green/photo bands (switches to light text + sunshine eyebrow).
